## üöÄ Running the project

```bash
# Clone the repository
$ git clone https://github.com/ImGabreuw/quote-card.git

# Enter the project directory
$ cd quote-card

# Install dependencies
$ npm i

# Run the application
$ npm run dev
```

Access http://localhost:3000/
